🏦 Bank Management System

A simple CLI-based banking system built using Python with JSON storage.

✨ Features
Create account with PIN validation.
Secure login (3 attempts + account lock).
Deposit & withdraw money.
Transfer money between users.
Transaction history with ID and timestamp.
Data saved using JSON file.

📂 Files
main.py
auth.py
models.py
storage.py
utils.py
full_project.pu
accounts.json

▶️ How to Run

Run the main file:
python full_project.py

ℹ️ Note
PIN must be 4 digits and should not be weak patterns.
Account locks for 5 minutes after 3 wrong attempts.
All transactions are saved automatically.



🎯 Practice of Python OOP, file handling, and basic authentication system
